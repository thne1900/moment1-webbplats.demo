Webbplats: Automatisering & publicering (Demo)

Uppgiftsbeskrivning:

Denna webbplats (demo) är ett moment i kursen "Frontend-baserad webbutveckling". Målsättningen med denna uppgift är att bli insatt kring användandet av NodeJS i skapandet av en webbplats. Om hur arbetsprocessen kan komma att underlättas vid tillämpning av NodeJS. Även att genom Parcel (kommandobaserat verktyg), göra uppgifter automatiserade via kommandon för webbplatsens utveckling. För att sedan kunna versionshantera till GitHub och därifrån publicera webbplatsen automatiskt via publiceringsplattformen Netlify.
